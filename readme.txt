#

Для запуска приложения обработки библиотеки из *.txt файлов

1. Поместите его в путь который есть в %GOHOME% или добавьте каталог проекта в %GOHOME%

2. В директории проекта запустите (или запустите собраный файл)
   $ ./go build main.go

3. В директорию /lib добавьте файлы *.txt которые на нужно обслуживать

4. Создайте пустую базу данных в MySQL

5. настройте settings.ini 
   - порт вебсервера 5051 (для теста)
   - dsn = admin:admin@tcp(127.0.0.1:3306)/base?charset=utf8&parseTime=True&loc=Local" (ту что вы создали ранее)
     (кстати лучше писать IP, с хостнеймом не очень заводится)
   - debug по желанию

6. Для просмотра конфига запустите
   $ ./main config

7. Запуск сервера обработки и API
   $ ./main run

8. Запуск GUI, перейдите в директорию /gui запустите
   $php yii serve --port 5050

9. В строке адреса вашего браузера напишите
   http://localhost:5050/
